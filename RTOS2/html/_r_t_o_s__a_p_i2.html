CMSIS-RTOS C API Version 2

Overview of all CMSIS-RTOS C API v2 functions.

  • Kernel Information and Control
    • osKernelInitialize : Initialize the RTOS kernel for creating objects.
    • osKernelStart : Start the RTOS kernel scheduler.
    • osKernelSuspend : Suspend the RTOS Kernel scheduler.
    • osKernelResume : Resume the RTOS Kernel scheduler.
    • osKernelState : Query if the RTOS kernel is inactive, running, or suspended.
    • osKernelTime : Get the RTOS kernel time (milli-second).
    • osKernelSysTick : Get RTOS kernel system timer counter.
    • osKernelSysTickFrequency : RTOS kernel system timer frequency in Hz.
    • osKernelSysTickMicroSec : Convert microseconds value to RTOS kernel system timer value.
  • Thread Management
    • osThreadNew : Create and start execution of a thread function.
    • osThreadTerminate : Stop execution of a thread function.
    • osThreadSuspend : Suspend a thread function.
    • osThreadResume : Resume a thread function.
    • osThreadGetState : Get current state of an active thread.
    • osThreadYield : Pass execution to next ready thread function.
    • osThreadGetId : Get the thread identifier to reference this thread.
    • osThreadSetPriority : Change the execution priority of a thread function.
    • osThreadGetPriority : Obtain the current execution priority of a thread function.
  • Generic Wait Functions
    • osDelay : Wait for a specified time.
    • osDelayUntil : Wait until a specified kernel time.
    • osEventWait : Wait for Thread Flags, Event Flags, Message, Mail, or Timeout.
  • Timer Management
    • osTimerNew : Create and define timer with related callback function.
    • osTimerDelete : Step a timer and remove related callback function.
    • osTimerStart : Start or restart the timer with a time value.
    • osTimerStop : Stop the timer.
    • osTimerIsRunning : Check if a timer is running.
  • Thread Flags Management Functions
    • osThreadFlagsSet : Set the specified Thread Flags of an active thread
    • osThreadFlagsClear : Clear the specified Thread Flags of an active thread
    • osThreadFlagsGet : Get the current Thread Flags of an active thread.
    • osThreadFlagsWait : Wait for one or more Thread Flags of the current running thread to become signaled
  • Event Flag Objects
    • osEventFlagsNew : Create and initialize an Event Flags object.
    • osEventFlagsDelete : Delete an Event Flags object.
    • osEventFlagsSet : Set the specified Event Flags.
    • osEventFlagsClear : Clear the specified Event Flags.
    • osEventFlagsGet : Get the current Event Flags.
    • osEventFlagsWait : Wait for one or more Event Flags to become signalled.
  • Mutexes
    • osMutexNew : Create and initialize a Mutex object.
    • osMutexDelete : Delete a Mutex object
    • osMutexAcquire : Obtain a Mutex or Wait until it becomes available.
    • osMutexRelease : Release a MMutex.
  • Semaphores
    • osSemaphoreNew : Create and initialize a Semaphore object
    • osSemaphoreDelete : Delete a Semaphore object.
    • osSemaphoreAcquire : Obtain a Semaphore token or Wait until it becomes available.
    • osSemaphoreRelease : Release a Semaphore token.

  • Memory Pool
    • osMemoryPoolNew : Define and initialize a fix-size memory pool.
    • osMemoryPoolAlloc : Allocate a memory block.
    • osMemoryPoolFree : Return a memory block to the memory pool.
    • osMemoryPoolGetInfo : Get a Memory Pool information
    • osMemoryPoolDelete : Delete a Memory Pool object
  • Message Queue
    • osMessageQueueNew : Create and initialize a message queue.
    • osMessageQueuePut : Put a message into a message queue.
    • osMessageQueueGet : Get a message or suspend thread execution until message arrives.
    • osMessageQueueGetInfo : Get a Message Queue information.
    • osMessageQueueReset : Reset a Message Queue to initial empty state.
    • osMessageQueueDelete : Delete a Message Queue object.
  • Mail Queue
    • osMailQueueNew : Create and Initialize a Mail Queue object.
    • osMailQueueAlloc : Allocate a memory block for mail from a mail memory pool.
    • osMailQueuePut : Put a Mail into a Queue.
    • osMailQueueGet : Get a Mail from a Queue or timeout if Queue is empty.
    • osMailQueueFree : Free a memory block by returning it to a mail memory pool.
    • osMailQueueGetInfo : Get a Mail Queue information.
    • osMailQueueReset : Reset a Mail Queue to initial empty state.
    • osMailQueueDelete : Delete a Mail Queue object.
