Delete a flag object. The function releases internal memory obtained for flag handling. After this call the flag ID is no longer valid and cannot be used. The flag may be created again using the function osEventFlagsNew.

Code Example

#include "cmsis_os.h"
osFlagDef my_flag; // Flag name definition
osFlagId my_flag_id; // Flag ID populated by the function osFlagCreate()
void DeleteFlag (osFlagId my_flag_id) {
osStatus status;
if (my_flag_id != NULL) {
status = osFlagDelete(my_flag_id);
if (status != osOK) {
// handle failure code
}
}
}

int32_t osEventFlagsWait ( osEventFlagsId  flags_id,
int32_t  flags,
uint32_t  options,
uint32_t  millisec 
)
Parameters
[in]flags_idevent flags ID obtained by osEventFlagsNew.
[in]flagsspecifies the flags to wait for.
[in]optionsspecifies flags options (osFlagsXxxx).
[in]millisecTimeout Value or 0 in case of no time-out.
Returns
event flags or error code if negative.

Suspend the execution of the current RUNNING thread until one or more flags are set for flag object specified by flag_id. When the flags that are waited for are already set, the function returns instantly.

The argument attributes can be used to specify whether to wait for any specified flag in the flags argument (default) or to wait for all flags to be set (use osFlagWaitForAll). Specifying osFlagAutoClear will automatically clear any specified flag in the flags argument (default) or all flags when used in conjunction with osFlagWaitForAll.

The argument millisec specifies how long the system waits for a message to become available. While the system waits, the calling thread is put into the state WAITING. The millisec timeout value can have the following values:

  • when millisec is 0, the function returns instantly.
  • when millisec is set to osWaitForever the function will wait for an infinite time until a message arrives.
  • all other values specify a time in millisecond for a timeout.
Note
The parameter millisec must be 0 for using this function in an ISR.
Interrupt Service Routines can call this function.

Status and Error Codes

  • osOK: no message is available in the queue and no timeout was specified.
  • osEventTimeout: no message has arrived during the given timeout period.
  • osEventMessage: message received, value.p contains the pointer to message.
  • osErrorParameter: a parameter is invalid or outside of a permitted range.

Code Example

#include "cmsis_os.h"
osFlagDef my_flag; // Flag name definition
osFlagId my_flag_id; // Flag ID populated by the function osFlagCreate()
void StartApplication (void) {
my_flag_id = osFlagCreate (osFlag(my_flag));
if (my_flag_id != NULL) {
event = osFlagWait(my_flag_id, 0x17, osFlagWaitForAll,0);
if (event != osOK) {
// handle failure code
} else {
// do something
}
} else {
// flag object not created
}
}
